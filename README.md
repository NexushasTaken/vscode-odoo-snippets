# odoo-snippets README



![demo](source/demo.gif)



* 包含 537 个代码片段

* 支持Camel Case 和 Snake Case 之间的 自动转换


例如

1. 模型名  ModelName  >  model.name
2. 字段名  field_name  >  Field Name
3. 记录名  view_model_name_viewtype  >   model.name.type.view



**Enjoy!**
